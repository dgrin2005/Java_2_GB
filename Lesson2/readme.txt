1) Файл с массивом "array.txt" должен находиться в одном каталоге с файлом Lesson2.class при запуске команды "java Lesson2"
2) Файл "array.txt" содержит массив в виде строк элементов, в каждой строке элементы должны быть разделены пробелом (по умолчанию).
3) В обработчиках сообщений об ошибках реализован вывод номеров строк и столбцов массива, принимая, что нумерация этих строк и столбцов начинается с 1 , а не с 0 (для удобства проверки).

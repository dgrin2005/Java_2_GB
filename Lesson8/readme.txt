Игра "Крестики-нолики" с возможностью играть по сети (клиент-серверный вариант)

1) Сервер можно запустить с параметром: 
java TicTacToeSrv <порт сервера>
По умолчанию берется порт 8189, в случае установки иного порта, у клиентов его также нужно менять через Меню - Настройки

2) Команды сервера (подаются из командной строки запущенного сервера):
set <размер карты> <количество точек для выигрыша> - установить параметры карты и определения выигрышной позиции
get - получить текущие параметры: размер карты, количество точек для выигрыша, порт сервера
exit - отключение (закрытие) сервера

3) Для запуска игры на стороне клиента нужно в меню выбрать:
  а) "Новая игра с человеком" - для игры по сети с соперником-человеком (необходимо подключение второго клиента)
  б) "Новая игра с AI" - для игры по сети с сервером

Замечания.
1) В игре с человека с человеком первым ходит тот кто подключился (начал игру) первым
2) В игре человека и AI первым ходит человек
3) В случае выбора первым игроком игры с AI, второй подключающийся игрок будет ожидать окончания сеанса первого
4) В случае выбора первым игроком игры с человеком, второй подключающийся игрок может начать только игру с человеком
5) Настройки в клиенте и на сервере можно поменять только вне сеанса игры

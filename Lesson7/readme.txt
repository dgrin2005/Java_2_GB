1) База данных пользователей создается при первом запуске сервера автоматически

2) Команды сервера:
add <user> <password> - добвавить нового пользователя user с паролем password в базу данных
remove <user> - удалить пользователя user из базы данных
list - отобразить список пользователей из базы данных (с паролями)
listc - отобразить спсисок подключенных на данный момент пользователей
exit - отключение (закрытие) сервера
/w <user> <message> - отправить сообщение message пользователю user
/a <message> - отправить сообщение всем подключенным пользователям

3) Команды клиента:
list - отобразить список пользователей из базы данных (без паролей)
listc - отобразить спсисок подключенных на данный момент пользователей
exit -  отключение (закрытие) клиента
/w <user> <message> - отправить сообщение message пользователю user
/a <message> - отправить сообщение всем подключенным пользователям

4) В случае неверного отображения русских букв в принятых сообщениях на любой стороне, необходимо предварительно в консолях (со стороны сервера и клиентов) запустить команду 'chcp 1251'
